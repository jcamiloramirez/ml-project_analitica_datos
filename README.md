## Descripción del Proyecto
El repositorio está diseñado para alojar el código y los experimentos correspondientes a las unidades del curso. Esto abarca desde la preparación y el análisis exploratorio de datos (EDA), hasta la implementación, entrenamiento y evaluación de modelos de aprendizaje supervisado y no supervisado.

## Estructura del Directorio
El entorno de trabajo sigue una estructura profesional para proyectos de Machine Learning:

* `data/`: Almacena los datos crudos (`raw/`) y los datos listos para el modelado (`processed/`). Por buenas prácticas, el contenido de estas carpetas no se sincroniza en el repositorio.
* `models/`: Archivos binarios de los modelos predictivos entrenados.
* `notebooks/`: Entornos interactivos (Jupyter Notebooks) utilizados para la exploración inicial y la experimentación.
* `reports/`: Documentos finales, incluyendo gráficas, tablas y el informe consolidado.
* `src/`: Código fuente modular en Python con los scripts principales de ejecución, preprocesamiento y modelado.
